# Shavkatjon Yuldashev — Data Science Portfolio

A collection of end-to-end data science projects demonstrating statistical modeling, machine learning pipelines, and analytical thinking. Each project follows a structured workflow: data loading → preprocessing → EDA → modeling → diagnostics → evaluation → business insights.

---

## Projects

| Project | Technique | Target | Key Result |
|---------|-----------|--------|------------|
| [GPA Prediction](./gpa-prediction/) | OLS Regression, Dummy Variables | Student GPA | Attendance increases predicted GPA by ~0.18 points independent of SAT score |
| [Used Car Price Prediction](./used-car-price/) | Linear Regression, Log Transform, Cross-Validation | Resale Price | R² = 0.82 on test set, 5-fold CV confirms stability |
| [Bank Marketing Classifier](./bank-marketing/) | Logistic Regression, ROC/AUC, Confusion Matrix | Term Deposit Subscription | 86% accuracy, AUC = 0.93, no overfitting |

## Tech Stack

**Languages:** Python 3.10+  
**Core Libraries:** NumPy, Pandas, Scikit-learn, Statsmodels, SciPy  
**Visualization:** Matplotlib, Seaborn  
**Environment:** Jupyter Notebook

## How to Run

```bash
# Clone the repository
git clone https://github.com/shavkatjony/data-science-portfolio.git
cd data-science-portfolio

# Install dependencies
pip install -r requirements.txt

# Open any project
cd gpa-prediction
jupyter notebook gpa_prediction.ipynb
```
